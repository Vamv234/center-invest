# Center-Invest Backend

Из PDF удалось достать стек проекта: `FastAPI`, `Redis`, `Docker`, `WebSockets`, фронт на `React/Next.js`.
В этой версии backend локальная база переведена на `SQLite`, чтобы модуль было проще запускать и подключать в командной разработке без отдельного PostgreSQL.
Этот модуль реализует backend-часть под командную разработку и изолирован в директории `backend/`.

## Что входит

- JWT-авторизация с access/refresh токенами
- хранение и отзыв пользовательских сессий
- профиль пользователя и личный кабинет
- трекинг прохождения сценариев, ошибок, попыток и процентов успешности
- рейтинг, лиги и прозрачная репутационная логика
- Redis-кэш для лидерборда
- SQLite-база в файле `data/center_invest.db`
- WebSocket-канал для обновления прогресса
- Alembic-миграция и Docker-инфраструктура

## Быстрый старт

1. Скопируйте `.env.example` в `.env`.
2. Запустите сервисы:

```bash
docker compose up --build
```

3. Примените миграции:

```bash
docker compose exec backend alembic upgrade head
```

4. Документация FastAPI будет доступна по адресу `http://localhost:8000/docs`.

SQLite-файл по умолчанию создаётся в `backend/data/center_invest.db`.

## Логика репутации

- регистрация: `+50`
- успешное прохождение сценария: `+25`
- бонус за высокий процент: `+0..10`
- бонус за улучшение личного рекорда: `+5`
- штраф за фарм одного и того же сценария после двух успешных прохождений: `-5`
- штраф за каждую ошибку: `-3` или больше, если фронт передал `penalty_points`
- провал сценария: `-15`
- итоговое изменение за попытку ограничивается диапазоном `[-40, +60]`
